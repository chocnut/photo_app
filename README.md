# Photo App

## Requirements

- NodeJS
- NPM
- Please copy and rename the .env.example to .env.local

## How to run the project?

```
- npm run dev
```

## How to run the unit test?

```
- npm run test:ci
```

## How to run the e2e?

```
- npm run cypress:headless
```

## Libraries used:

- Next JS
- React Query (To handle fetching, pre-fetching and data )
- Jest (unit testing)
- Cypress (e2e)
